Pricing & Plans

Parseflow keeps pricing simple and transparent.

Starter monthly — $9/month​

  • 2,000 API requests per month
  • 20 requests per minute rate limit
  • Up to 10 MB per upload
  • Async jobs and batch processing
  • Deterministic processing
  • BYOK path available
  • Get Started →

Growth monthly — $24/month​

  • Better value for regular usage
  • Same core API access as Starter
  • Better fit for teams using Parseflow often
  • Same request and upload limits
  • Same async/batch access
  • BYOK path available
  • Get Started →

BYOK option​

If you already have your own model-provider key, you can use advanced extraction through mode=byok_assisted.

This is the best path for users who want more control over model usage without relying on Parseflow-managed inference.

Current limits​

The current API limits are documented in Limits.

At launch, the main split is:

  • deterministic usage through your Parseflow API key
  • advanced extraction through BYOK
  • async and batch features on paid usage

Which plan should I choose?​

  • Choose Starter monthly if you want the simplest low-cost way to begin.
  • Choose Growth monthly if you expect regular usage and want the better value plan.
  • Choose BYOK if you already have provider access and want advanced extraction with your own key.

Billing note​

The public checkout flow is documented in Billing. That is where you buy access and receive your API key.
